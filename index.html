Abstract

This project is a basic website created with HTML, CSS, and JavaScript. It introduces a simple project archive page inspired by my interest in environmental systems, ocean data, and computational design (>^ω^<)

Motivation

I am interested in how websites can become more than static pages. They can act as interfaces for research, storytelling, and interaction. This small exercise helps me understand how structure, style, and behavior work together on the web~~~~~

Methodology

The page uses HTML to organize the content, CSS to control the visual style, and JavaScript to add a simple interactive button. Together, these three files demonstrate a basic web development workflow(｡ì _ í｡)
