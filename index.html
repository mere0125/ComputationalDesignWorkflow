Abstract

This project modifies a basic HTML, CSS, and JavaScript website template into a Webcore-inspired digital archive. The page uses nostalgic internet visuals, browser-window composition, bright colors, and small interactive moments to explore the feeling of an early personal website(^_−)−☆

Motivation

I wanted the website to feel less like a clean corporate webpage and more like a strange personal internet artifact. Webcore is interesting to me because it turns old digital interfaces, pop-ups, pixel graphics, and amateur web design into an aesthetic language(｡ì _ í｡)

Methodology

I kept the original website structure from the class template, but changed the visual system through color, typography, borders, window-like panels, pixel details, and JavaScript interaction. The goal was to practice AI-assisted “vibe coding” while still understanding how each file works（＾∇＾）
